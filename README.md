# YDTA-PROGECT
# 📌 Project Overview

This project focuses on analyzing a financial transactions dataset using Python (pandas).
The goal is to clean, explore, and visualize financial data to uncover insights.

# 📁 Dataset
File: finance_transactions_dataset.xlsx
Rows: 1,000
Columns: 14


# The dataset contains:

1. Transaction details (Amount, Tax, Net Amount)
2. Company and department info
3. Currency types (USD, GBP, EUR, NGN)
4. Approval status
5. Dates (2021–2024)

   
# 🧰 Tools & Libraries
1. Python 3.8+
2. pandas
3. matplotlib
4. openpyxl


# Install dependencies:

pip install pandas matplotlib openpyxl notebook



# 🚀 Features / What This Project Does

1.Load and inspect Excel data
2. Handle missing values
3. Convert text to datetime
4. Remove duplicates
5. Perform grouping and aggregation
6. Create visualizations using .plot()



# 📊 Visualizations Included
1. Bar chart → Transaction Type vs Total Amount
2. Line chart → Monthly trends
3. Pie chart → Transaction distribution
4. Histogram → Data distribution
5. Scatter plot → Relationships



# 📈 Key Learning Outcomes
Data cleaning in pandas
Grouping and aggregation
Financial data interpretation
Quick visualization using pandas .plot()
